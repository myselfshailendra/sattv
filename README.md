# SatTV

Direct to home (D2H) operator as a mini software.

### Steps to Run The Program:

1- Unzip the repo


2- Change the directory.

```
    cd sattv
```

3- install gems.

```
    bundle install
```

4- Run tests.

```
    $ rspec spec
```

5- Run the program.

```
    $ ruby start_sattv.rb
```
